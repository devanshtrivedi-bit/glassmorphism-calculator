# 🧮 Cyberpunk Glassmorphism Calculator

A beautiful, fully functional scientific calculator featuring an ultra-premium Glassmorphism UI, an animated liquid aurora effect, and native Web Audio sound effects.

## ✨ Features

**[🚀 Play with the Live Demo Here!](https://raw.githack.com/devanshtrivedi-bit/glassmorphism-calculator/main/index.html)**

- **Stunning UI/UX**: Frosted glassmorphism design with a dynamic, glowing internal aurora animation.
- **Scientific Engine**: Powered by `math.js` to handle complex equations, order of operations (BODMAS), and scientific functions (sin, cos, log, etc.).
- **Haptic & Audio Feedback**: Custom-built "tick" sounds using the native Web Audio API for a satisfying mechanical feel on every press.
- **Full Keyboard Support**: Seamlessly use your numpad and keyboard shortcuts (e.g., `s` for sine, `p` for pi, `Enter` for equals).
- **Responsive**: Perfectly scaled for both desktop and mobile displays.

## 🛠️ Technologies Used

- **HTML5**: Semantic structure.
- **Vanilla CSS3**: Advanced pseudo-elements, dynamic keyframe animations, glassmorphism filtering (blur/saturate).
- **Vanilla JavaScript**: Logic, event handling, and DOM manipulation.
- **Math.js**: Powerful expression evaluation.
- **Web Audio API**: Real-time sound generation without external audio files.

## 🚀 How to Run Locally

1. Clone this repository:
   ```bash
   git clone https://github.com/devanshtrivedi-bit/glassmorphism-calculator.git
   ```
2. Navigate into the directory:
   ```bash
   cd glassmorphism-calculator
   ```
3. Open `index.html` in your web browser. Or, for the best experience, run a local live server (e.g., VS Code Live Server or Python HTTP Server).
